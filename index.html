DOM Manipulation Notes

What is DOM Manipulation?
The Document Object Model (DOM) refers to a way to represent a document, or a web page, within a programming language, typically JavaScript. This allows you to access the styling, structure, or content of a document, updating it with the use of logic you couldnt get otherwise.

This information is modeled as an Object, hence the name, converting the contents of the documents into values that are able to be manipulated using variables and methods.
